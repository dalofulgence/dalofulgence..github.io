Reading Notes

object is a self-contained set of related values and functions
Ex: const superman = { name: 'Superman', 'real name': 'Clark Kent', height: 75, weight: 235, hero: true, villain: false, allies: ['Batman','Supergirl','Superboy'], fly() { return 'Up, up and away!'; } };
Acessing properties: superman.name or superman['name']= 'Superman'

Finding all properties: for(const key in superman) { console.log(key + ": " + superman[key]); } << "name: Superman" << "real name: Clark Kent" << "height: 75" << "weight: 235" << "hero: true" << "villain: false" << "allies: Batman,Supergirl,Superboy" << "fly: function (){ console.log(\"Up, up and away!\"); }"

accessing keys: for(const key of Object.keys(superman)) { console.log(key);

Accessing values: for(const value of Object.values(superman)) { console.log(value);

Adding properties: superman.city = 'Metropolis';
